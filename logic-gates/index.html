Logic gates


Logic gates are a form of logic in which "gates" return true/false values depending on the combination of two true/false values provided as "inputs" to the logic gate. Transisters physically let electricity flow according to the rules of these conceptual logic "gates", using binary logic to represent True/1 or False/0, with a flow of electricity representing 1, and no electrical flow representing 0.

The main three logic gates:

  • OR || - True if one or both are true
  • NOT ! - True if input is false
  • AND && - True if both inputs are true

Truth table for AND:

A|B|Output

  • NOR - not or: false if either or both of the inputs are true
  • NAND - Not and: False if both inputs are true, otherwise true
  • XOR - Exclusive or: true if one of them is true, False if both inputs are true or false
  • XNOR - Exclusive not or: true if either none or both of the inputs are true, false if one of the inputs is true
  • BUFFER - Buff up the input

Relational and logical operators: || && ! <= >= ==.
