Digital insights course

Digital Insights was a course I did with Exeter College from January to March of 2021 designed to provide insight into working in software development. I learnt about different software development methodologies (how teams are organised to produce software), as well as the thinking processes behind producing software, such as logic concepts like syllogisms, and problem-solving strategies.

Throughout the course we produced a blog that recorded what we learnt, and there was a final assignment in which we would create a plan for a software project and present this. Unfortunately I didn't complete the blog or the final assignment due to health issues, however I did create some of the blog which can be seen at https://miles-harvey.tiiny.site.

Software projects I’ve created

Guix

  • Guix Channel: Non-free software – This is a Guix channel (repository of package definitions) containing software that doesn’t meet the standards of the Guix project’s software license criteria for free software, such as software that doesn’t allow the source to be shared.
  • Guix channel: Free software – This is a Guix channel containing software that meets the criteria for submission into the Guix project, but that I haven’t yet submitted.
  • Guix System configurations – My configuration files used to build my laptop’s Guix system.
  • Guix Libfinder – A script for finding the libraries a binary looks for, and lists the packages that contain these required binaries.
  • Guix module creator – A script for generating a package definition for Guix.
